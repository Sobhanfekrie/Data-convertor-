## English Version

### Overview
Data Converter is a powerful tool for converting and processing files. It leverages the speed of **C** for heavy data processing, the flexibility of **Python** for handling file operations and user interface, and a minimal **PHP** interface (~3%) to display output via a web browser.

### Features
- Convert any type of data file.
- Fast processing with C modules.
- Easy-to-use interface via Python CLI.
- Optional PHP web interface to view output.
- Input and output file handling.

### Installation
1. Clone the repository:
   ```bash
   git clone <repository-url>

2. Compile the C program:

cd src
gcc -o converter converter.c


3. Run the Python interface:

python main.py


4. Optional: Place web/api.php on a PHP server to view output in the browser.



Usage

Put your input file in the input/ folder.

Run the Python script to process it.

Check the output in output/.

Open web/api.php in a browser to view output content.
